3. Выделить истинные выражения:


T потоки в Python выполняются в рамках одного процесса, разделяя GIL
выполнением Python потоков в Linux занимается процесс, в котором они запущены
T выполнением Python потоков в Linux занимается операционная система
у каждого потока есть собственная память
T все потоки разделяют память процесса в котором они созданы

Функции, занимающиеся вычислениями на CPU
